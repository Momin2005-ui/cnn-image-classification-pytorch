# CNN Image Classification using PyTorch

## Overview
This project implements a Convolutional Neural Network (CNN) from scratch for multiclass image classification using PyTorch.

## Features
- Custom CNN architecture
- ReLU activation
- MaxPooling
- CrossEntropyLoss
- Adam optimizer
- Train-validation split

## Architecture
Conv → ReLU → MaxPool  
Conv → ReLU → MaxPool  
Flatten → Fully Connected → Output

## How to Run

```bash
pip install -r requirements.txt
python train.py
